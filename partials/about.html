O que é o projeto De Olho Neles

De olho neles é um projeto que tenta mostrar para o povo brasileiro onde os deputados gastam o seu cartão de crédito corporativo (CEAP).

Qual o salário de um deputado?

De acordo com o site da câmara de deputados o salário de um deputado é de R$ 33.763.00.

O que é CEAP?

A Cota para Exercício da Atividade Parlamentar (CEAP) é um valor usado para ressarcir deputados com qualquer despesa feita em razão do trabalho, em situações onde não cabe abrir uma licitação.

Se compararmos com empresas privadas podemos dizer que CEAP é o cartão corporativo que um funcionário tem direito de gastar em alimentação, transporte e desenvolvimento pessoal, etc.

Porém, o valor do CEAP varia de R$ 30.788,66 até R$ 44.632,46 para cada deputado. O que em muitos casos é maior que o próprio salário do deputado. Além disso, se você é brasileiro, quem paga esta conta é você.

Como estes dados são obtidos?

De acordo com o Ato da mesa de nº 43, de 21/05/2009 o reembolso dos valores está condicionado a declaração de onde e como estes valores foram usados. Estes dados se encontram no próprio portal do CEAP.

Porém a manipulação deles só se faz possível graças ao trabalho que a Operação Serenata de Amor fez para permitir o entendimento dos dados de forma clara.

Como essa ideia surgiu?

A ideia surgiu após o conhecimento na Operação Serenata de Amor que tenta automatizar a detecção de fraudes desta verba, junto com o necessidade de mostrar para a população uma pequena parte das despesas públicas.
